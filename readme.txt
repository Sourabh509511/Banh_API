First Fulfill all requirements given in requirements.txt file.


To get data from database using:

    1.IFSC code of the branch:
            Add '?ifsc="ifsc_code_of_branch_you_want_details"' after the urls of the site.

                syntax:http://URL/?ifsc=''

    2.BANK NAME and CITY of branches:
            Add '?bank_name="bank_name"&city="city"' after the urls of the site.

                syntax:http://URL/?bank_name=''&city=''
